# US-7: Countdown Beeps

**As** Thomas
**I want** audio beeps at the end of each phase
**So that** I know when to transition without looking at the screen

## Acceptance Criteria

- [ ] 3 beeps in the last 3 seconds of each phase (1 beep per second)
- [ ] Beeps are distinct and audible over outdoor noise
- [ ] Final beep at 0 seconds signals phase transition
- [ ] Final beep should be slightly longer in duration and louder
